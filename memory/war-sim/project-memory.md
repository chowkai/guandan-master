## 📋 项目概述

**2D 战略战争游戏平台**，模拟从冷兵器到热兵器的所有战争。

**核心特色**:
- 🎯 纯军事指挥视角（不能直接指挥个体）
- 🌍 大规模战斗（10 万 + 单位）
- 👤 单位全生命周期（生老病死、履历记录）
- 📦 后勤为核心（物资、运输、补给）
- 🛠️ 平台化（装备/地图/战役编辑器）

## 📝 重要讨论记录

### 斥候自主系统 (v1.7)
- 斥候和运输兵一样，完全自主工作
- 不需要玩家微操，配置后自动执行
- 侦察细致度 = 配置的侦察兵力
- 斥候遭遇战自动处理

### 单位全生命周期 (2026-03-21 补录)

**个体行为三原则**:
1. **生存第一**: 饥渴/受伤/力竭时会优先确保生存
   - 补给耗尽 → 惊慌度上升 → 士气下降 → 逃跑/投降
   - 斥候/传令兵有特殊技能减免此效果
2. **依附编制**: 听从号令，借助集群力生存
   - 溃散超出视距/声距 → 随机乱跑
   - 遭遇敌方 → 俘虏；遭遇己方补给点 → 收容
3. **渴望晋升**: 根据能力与军功自动提拔

**编制系统**:
- 多层级架构（班/排/连/营/...）
- 玩家指挥编制，不能直接指挥个体
- 命令通过传令兵传递（非即时生效）
- 编制自动补充人员/装备/物资

**传令兵机制**:
- 玩家可调整命令紧急度
- 自动多派分时分路传递
- 可能被拦截（意外事件）

**群体崩溃**:
- 凝聚力溃到一定程度 → 群体性崩溃
- 非个体单独逃跑

**数据可视**:
- 只显示表面数据（士气、补给等）
- 隐藏深层数据（惊慌度、个体属性等）

**装备系统细节**:
- 投掷物：射程、重量、破甲度、爆炸范围
- 非投掷：尖锐度、对护甲伤害
- 折旧度 + 可修复（工匠/维修厂）
- 缴获装备可用（需技能/运输能力）
- 玩家可在编辑器 DIY 任何装备（包括原子弹）

**经济系统**:
- 抽象层只有资金概念
- 所有物资以物质形式存在（重量、体积）
- 地域有人口与经济程度 → 决定税收/征兵
- 不想太复杂（无外交系统）

**建筑系统**:
- 经济建筑也可移动（需付出迁移代价）
- 可破坏、可占领
- 建筑内有平民（大部分被俘，少数忠诚者可能逃跑/反抗）

**平台定位**:
- 创造战争世界的平台
- 官方 MOD 计划：红军战史、三国战史
- 时代范围由地图/战役编辑器决定
- 玩家复现历史上所有战争

## 🎮 核心三问答案

### A. 乐趣点
✅ **战略布局 + 人性洞察 + 历史沉浸感**
- "运筹帷幄之中，决胜千里之外"的成就感
- "见证历史，改写历史"的沉浸感

### B. 指挥边界
✅ **玩家可指挥任意编制**（包括最底层的班或经济建筑）
- 不能直接指挥个体
- 命令通过传令兵传递
- 经济建筑也可移动（矿场只能在矿脉上）

### C. 时代范围
✅ **由地图/战役编辑器决定**
- 不攀科技树
- 装备/通讯/马匹从经济建筑产出
- 士兵需要训练时间和资金掌握技能
