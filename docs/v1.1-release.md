### 5. 报牌提醒优化 (P1) ✅

**改进内容**:
- ✅ 移除剩余牌数常显
- ✅ 只剩≤6 张时提醒一次
- ✅ 提醒时播放音效
- ✅ 每个玩家独立提醒标记

**修改文件**:
- `src/client/js/game.js`

**逻辑**:
```javascript
if (count <= 6 && !this.hasReminded[position]) {
    this.hasReminded[position] = true;
    this.showGameMessage(`${playerName} 剩余 ${count} 张牌！`);
    this.playSound('hint');
}
```
