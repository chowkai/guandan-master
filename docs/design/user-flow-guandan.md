## 📊 整体流程

```mermaid
graph TD
    A[打开小程序] --> B{已登录？}
    B -->|否 | C[登录页]
    B -->|是 | D[首页]
    C --> D
    D --> E[创建房间]
    D --> F[加入房间]
    D --> G[快速匹配]
    E --> H[房间页]
    F --> H
    G --> H
    H --> I[游戏页]
    I --> J[结算页]
    J --> K{继续游戏？}

    K -->|否 | D
```

## 🎴 游戏流程

```mermaid
graph TD
    A[房主开始游戏] --> B[发牌动画]
    B --> C[玩家抓牌]
    C --> D{轮到自己？}
    D -->|否 | E[等待其他玩家]
    D -->|是 | F[选牌]
    F --> G{出牌？}
    G -->|出牌 | H[验证牌型]
    G -->|不出 | I[跳过]
    H --> J{牌型正确？}
    J -->|是 | K[出牌成功]
    J -->|否 | L[提示错误]
    K --> M{有人出完？}
    M -->|否 | N[下一个玩家]
    M -->|是 | O[游戏结束]
    N --> D
    O --> P[结算]
```

## ⚠️ 边界情况处理

### 网络异常
```
出牌时断网 → 提示"网络异常" → 自动重连 → 恢复游戏状态
```

### 玩家退出
```
游戏中退出 → 判负 → 托管继续 → 游戏结束
```

### 超时处理
```
轮到自己超时 → 自动托管 → 出最小牌
```
