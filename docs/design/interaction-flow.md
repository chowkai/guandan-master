## 详细流程 3: 暂停/继续

```
┌─────────────┐
│ 游戏进行中  │
└──────┬──────┘
       │ 点击"暂停"按钮
       ▼
┌─────────────┐
│ 显示暂停    │
│ 遮罩层      │
└──────┬──────┘
       │
       ├──────────────┐
       │ 继续游戏     │ 退出牌局
       ▼              ▼
┌─────────────┐  ┌─────────────┐
│ 关闭遮罩层  │  │ 保存牌局    │
│ 恢复游戏    │  │ 状态到本地  │
└──────┬──────┘  └──────┬──────┘
       │                │
       ▼                ▼
┌─────────────┐  ┌─────────────┐
│ 游戏进行中  │  │ 返回主界面  │
└─────────────┘  └──────┬──────┘
                        │
                        ▼
                 ┌─────────────┐
                 │ 主按钮变为  │
                 │ "继续游戏"  │
                 └──────┬──────┘
                        │
                        │ 点击"继续游戏"
                        ▼
                 ┌─────────────┐
                 │ 读取本地    │
                 │ 保存的牌局  │
                 └──────┬──────┘
                        │
                        ▼
                 ┌─────────────┐
                 │ 恢复游戏    │
                 │ 界面        │
                 └─────────────┘
```

### 暂停状态保存内容

```
保存的数据:
├── 当前局数
├── 四家手牌
├── 已出的牌
├── 当前回合（轮到谁）
├── 比分
├── 游戏模式（单机/P2P）
└── 时间戳
```

## 异常流程处理

### 1. 出牌不符合规则

```
用户选牌 → 点击出牌 → 验证失败
                              │
                              ▼
                    ┌─────────────┐
                    │ 手机震动    │
                    │ (Haptic)    │
                    └──────┬──────┘
                           │
                           ▼
                    ┌─────────────┐
                    │ 红色提示    │
                    │ "不符合规则"│
                    └──────┬──────┘
                           │
                           ▼
                    ┌─────────────┐
                    │ 自动撤销    │
                    │ 选中的牌    │
                    └─────────────┘
```

### 2. AI 出牌超时

```
AI 思考 → 超过 500ms 未出牌
                    │
                    ▼
          ┌─────────────┐
          │ 显示"思考中"│
          │ 动画        │
          └──────┬──────┘
                 │
                 ▼
          ┌─────────────┐
          │ 强制 AI 出牌 │
          │ (随机合法牌)│
          └─────────────┘
```

### 3. 应用被切到后台

```
游戏进行中 → 用户切出应用
                    │
                    ▼
          ┌─────────────┐
          │ 自动暂停    │
          │ 保存状态    │
          └──────┬──────┘
                 │
                 ▼
          ┌─────────────┐
          │ 应用后台    │
          │ 运行        │
          └──────┬──────┘
                 │
                 │ 用户切回应用
                 ▼
          ┌─────────────┐
          │ 自动恢复    │
          │ 游戏界面    │
          └─────────────┘
```

### 4. P2P 断线重连

```
游戏进行中 → 网络断开
                    │
                    ▼
          ┌─────────────┐
          │ 显示"重连中"│
          │ 遮罩层      │
          └──────┬──────┘
                 │
                 ├──────────────┐
                 │ 30 秒内重连  │ 超过 30 秒
                 │ 成功         │
                 ▼              ▼
          ┌─────────────┐  ┌─────────────┐
          │ 恢复游戏    │  │ 显示"游戏"  │
          │ 继续        │  │ 已结束"     │
          └─────────────┘  └─────────────┘
```

## 交互验收清单

| 流程 | 验收标准 | 状态 |
|------|----------|------|
| 开始游戏 | ≤2 次点击进入牌局 | ✅ |
| 发牌动画 | ≤3 秒完成 | ✅ |
| 出牌操作 | 选牌→出牌流畅 | ✅ |
| AI 响应 | ≤300ms 出牌 | ✅ |
| 暂停功能 | 随时暂停，状态保存 | ✅ |
| 继续功能 | 立即恢复，状态完整 | ✅ |
| 游戏结束 | 结算清晰，再来一局便捷 | ✅ |
| 异常处理 | 震动提示 + 文字说明 | ✅ |
