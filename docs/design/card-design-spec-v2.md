## 📐 基础规格

- **圆角**: 8px

### 边距规范
- 外边距：4px
- 内边距：6px

## 🎨 花字布局修正（v2.0 关键变更）

### 左上角区域
```
┌─────────┐
│    字    │  ← 28px，居中
│    🂡    │  ← 花色图标 20px
│         │
└─────────┘
```
- **排列方式**: 竖排（字在上，花在下）
- **文字**: 28px，黑色 (#000000)
- **花色图标**: 20px，红色 (♥♦) 或黑色 (♠♣)
- **位置**: 距左边 8px，距上边 8px

### 中间花色区域
```
         ┌─────────┐
         │         │
    ┌────┤    🂡    │  ← 偏右下
    │    │         │
    │    └─────────┘
    │    
┌───┴───┐
│字 + 花 │  ← 左上角
└───────┘
```
- **位置**: 卡牌中心偏右下
- **坐标**: 中心点向右偏移 8px，向下偏移 4px
- **花色图标**: 48px，红色 (♥♦) 或黑色 (♠♣)
- **目的**: 给左上角花字留出空间

### 右下角区域
```
┌─────────┐
│         │
│         │
│    🂡    │  ← 仅花色图标
│    字    │  ← 28px，居中
└─────────┘
```
- **排列方式**: 竖排（花在上，字在下）
- **文字**: 28px，黑色 (#000000)
- **花色图标**: 20px
- **注意**: v2.0 版本右下角**不需要花字组合**，仅显示倒置的花色 + 字

## 🃏 Joker 设计规范

### 小王 (Little Joker)
```
┌─────────────────────┐
│  ╔═══════════════╗  │
│  ║               ║  │
│  ║      👤       ║  │  ← 人物剪影 60px
│  ║               ║  │
│  ║     小王      ║  │  ← 28px 蓝色
│  ║               ║  │
│  ╚═══════════════╝  │
└─────────────────────┘
```

**视觉规格**:
- **背景**: 浅蓝色渐变 (#e3f2fd → #bbdefb)
- **图标**: 👤 人物剪影，60px，居中
- **文字**: "小王"，28px，蓝色 (#1976d2)
- **边框**: 蓝色 2px (#1976d2)
- **圆角**: 8px

### 大王 (Big Joker)
```
┌─────────────────────┐
│  ╔═══════════════╗  │
│  ║               ║  │
│  ║      👑       ║  │  ← 皇冠 60px
│  ║               ║  │
│  ║     大王      ║  │  ← 28px 红色
│  ║               ║  │
│  ╚═══════════════╝  │
└─────────────────────┘
```

**视觉规格**:
- **背景**: 浅红色渐变 (#ffebee → #ffcdd2)

- **边框**: 红色 2px (#d32f2f)
- **圆角**: 8px
