## 2. 花字布局设计（修正版）

### 整体布局结构

```
┌────────────────┐
│ A              │ ← 左上角：字 + 花（竖排）
│ ♥              │
│                │
│                │
│         ♥      │ ← 中间：大花色（偏右下）
│                │
│                │
│                │ ← 右下角：留空（无花字）
└────────────────┘
```

### 左上角（主识别区）✅

| 元素 | 规格 | 位置 |
|------|------|------|
| **字** | 24px, 600 | 距左 8px, 距上 8px |
| **花** | 20px × 20px | 字**下方**，间距 2px |
| **排列** | **竖排** | 字在上，花在下 |
| **颜色** | 红桃/方片：#d32f2f<br>黑桃/梅花：#212121 | 根据花色 |

**设计说明**:
- 字 + 花竖排，紧密排列时也能清晰辨识
- 左上角是主要识别区，符合用户阅读习惯

### 中间（辅助识别区）✅ 修正

| 元素 | 规格 | 位置 |
|------|------|------|
| **大花色图标** | 40px × 40px | **偏右下** |
| **水平位置** | 中心线偏右 10px | `left: 50% + 10px` |
| **垂直位置** | 中心线偏下 10px | `top: 50% + 10px` |
| **颜色** | 红桃/方片：#d32f2f<br>黑桃/梅花：#212121 | 根据花色 |

**设计说明**:
- 中间花色偏右下，给左上角花字留出空间
- 避免与左上角视觉冲突
- 作为辅助识别，尺寸可略小

### 右下角 ✅ 修正

| 元素 | 规格 | 位置 |
|------|------|------|
| **无花字** | - | 留空 |

**设计说明**:
- 右下角不需要花字（之前理解错误）
- 保持简洁，避免视觉干扰

## 3. 卡牌详细设计图

### 红桃 A 示例

```
┌────────────────────────────┐
│ A                          │
│ ♥                          │
│                            │
│                            │
│                            │
│              ♥             │
│                            │
│                            │
│                            │
│                            │
└────────────────────────────┘

尺寸标注:
- 卡牌：80×112px
- 左上角字：24px，距左上角各 8px
- 左上角花：20×20px，字下方 2px
- 中间花色：40×40px，位置 (50px, 66px) 即偏右下
```

### 紧密排列效果

```
手牌区横向滚动（27 张牌）:
[♠A]←12px→[♠K]←12px→[♠Q]←12px→[♠J]←12px→...

紧密排列时，左上角花字清晰可辨:
┌────┐ ┌────┐ ┌────┐
│A   │ │K   │ │Q   │
│♥   │ │♠   │ │♦   │
│    │ │    │ │    │
│    │ │    │ │    │
│  ♥ │ │  ♠ │ │  ♦ │
└────┘ └────┘ └────┘
 ↑左上角花字竖排，即使紧密排列也能看清
```

## 4. Joker 卡牌设计

### 需求回顾（PRD v1.0/v2.0）

**Joker 特殊要求**:
1. 区分大王/小王
2. 视觉上有辨识度
3. 符合扑克牌习惯

### 设计方案

#### 小王

```
┌────────────────────────────┐
│                            │
│         👤                 │
│                            │
│        小王                │
│                            │
│                            │
│                            │
│                            │
│                            │
└────────────────────────────┘

设计规格:
- 背景：浅蓝色渐变 (#e3f2fd → #bbdefb)
- 图标：👤 人物剪影（60px）
- 文字："小王" 28px，蓝色 (#1976d2)
- 边框：蓝色 2px
```

#### 大王

```
┌────────────────────────────┐
│                            │
│         👑                 │
│                            │
│        大王                │
│                            │
│                            │
│                            │
│                            │
│                            │
└────────────────────────────┘

设计规格:
- 背景：浅红色渐变 (#ffebee → #ffcdd2)
- 图标：👑 皇冠（60px）
- 文字："大王" 28px，红色 (#d32f2f)
- 边框：红色 2px
```

### Joker 设计规范

| 元素 | 小王 | 大王 |
|------|------|------|
| **背景色** | #e3f2fd → #bbdefb | #ffebee → #ffcdd2 |
| **图标** | 👤 60px | 👑 60px |
| **文字** | "小王" 28px | "大王" 28px |
| **文字颜色** | #1976d2（蓝色） | #d32f2f（红色） |
| **边框** | 2px #1976d2 | 2px #d32f2f |
| **尺寸** | 80×112px（同普通牌） | 80×112px（同普通牌） |
