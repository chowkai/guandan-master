## 🃏 手牌区设计规范（v5.0 核心变更）

### 多列网格布局

#### 布局结构
```
┌───────────────────────────────┐
│ ♠列  ♥列  ♦列  ♣列  王列      │ ← 列标题（可选）
├───────────────────────────────┤
│ A   A   A   A   大王           │ ← 第 1 行
│ K   K   K   K   小王           │ ← 第 2 行
│ Q   Q   Q   Q                  │ ← 第 3 行
│ J   J   J   J                  │ ← 第 4 行
│ 10  10  10  10                 │ ← 第 5 行
│ 9   9   9   9                  │ ← 第 6 行
└───────────────────────────────┘
```

#### 技术规格
| 参数 | 数值 | 说明 |
|------|------|------|
| **列数** | 5 列 | 4 花色 + 1 王列 |
| **最大行数** | 6 行 | 27 张牌÷5≈5.4，向上取整 |
| **卡牌尺寸** | 80×112px | 保持 v4.0 规格 |
| **列间距** | 12px | 紧凑排列 |
| **行间距** | -20px | 负值，轻微重叠（类似扇形） |
| **总宽度** | 448px | 5×80 + 4×12 |
| **总高度** | 572px | 6×112 + 5×(-20) |
| **容器位置** | (736, 750) | 底部中央，距底边 190px |

#### 卡牌分组规则
- **黑桃列 (♠)**: 所有黑桃牌，按 A-K-Q-J-10-9-8-7-6 排序
- **红心列 (♥)**: 所有红心牌，按 A-K-Q-J-10-9-8-7-6 排序
- **方块列 (♦)**: 所有方块牌，按 A-K-Q-J-10-9-8-7-6 排序
- **梅花列 (♣)**: 所有梅花牌，按 A-K-Q-J-10-9-8-7-6 排序
- **王列**: 大王在上，小王在下（固定顺序）

## 🎯 操作按钮区

### 按钮布局
```
┌─────────────────────────────────┐
│   [💡 提示]  [✅ 出牌]  [❌ 不出]   │
└─────────────────────────────────┘
```

### 技术规格
| 参数 | 数值 |
|------|------|
| **位置** | (960, 1010) - 手牌区下方 38px |
| **按钮尺寸** | 120×40px |
| **按钮间距** | 16px |
| **总宽度** | 392px (3×120 + 2×16) |
| **总高度** | 40px |

### 按钮样式
| 按钮 | 背景色 | 文字 | 图标 |
|------|--------|------|------|
| **提示** | 蓝色渐变 (#1976d2 → #42a5f5) | 16px 白色 | 💡 |
| **出牌** | 绿色渐变 (#388e3c → #66bb6a) | 16px 白色 | ✅ |
| **不出** | 红色渐变 (#d32f2f → #ef5350) | 16px 白色 | ❌ |

### 按钮状态

- **按下**: 亮度降低 20%，缩小 0.95 倍
- **禁用**: 灰色 (#9e9e9e)，不可点击

## 🎭 动画效果

### 选牌动画
- **触发**: 点击卡牌
- **持续时间**: 150ms
- **效果**: 黄色边框渐显 + 亮度提升
- **缓动**: ease-out

### 出牌动画
- **触发**: 点击 [✅ 出牌] 按钮
- **持续时间**: 300ms
- **效果**: 选中的牌飞向出牌区，缩放至 0.8 倍
- **缓动**: ease-out

### 理牌动画
- **触发**: 点击 [💡 提示] 按钮
- **持续时间**: 500ms
- **效果**: 所有牌按花色 + 大小排序，飞入对应列
- **缓动**: ease-in-out

### 胜利动画
- **触发**: 游戏胜利
- **持续时间**: 2000ms
- **效果**: 彩带飘落 + 所有手牌翻转展示
- **缓动**: ease-out

## ✅ v5.0 变更清单

### 问题：手牌区横向滚动体验差
- **用户反馈**: 需要拖来拖去选牌，不能一目了然
- **解决方案**: 多列网格布局（5 列×6 行）
- **效果**: 
  - ✅ 所有 27 张牌同时可见（572px 高度）
  - ✅ 直接点击选牌，无需拖动
  - ✅ 同花色自动归列，符合理牌习惯

### 技术实现
- **布局**: CSS Grid 或 Flexbox 实现 5 列网格
- **排序**: 后端按花色 + 大小排序，前端按序渲染
- **交互**: 点击选中、悬停高亮、长按锁定
- **动画**: 选牌/出牌/理牌动画优化

### 对比 v4.0
| 维度 | v4.0（横向） | v5.0（网格） | 改进 |
|------|-------------|-------------|------|
| **可见性** | 需滚动 | 全可见 | ⭐⭐⭐ |
| **操作性** | 拖来拖去 | 直接点击 | ⭐⭐⭐ |
| **空间** | 横向 2000px+ | 纵向 572px | ⭐⭐ |
| **理牌** | 手动整理 | 自动分组 | ⭐⭐⭐ |
