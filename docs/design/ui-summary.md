## 设计原则回顾

### 1. 深绿色牌桌

**主色调**: `#1a472a` (经典赌场牌桌绿)

```
色彩比例:
████████████████████████████  深绿色 60%
████████████  白色 25%
██████  浅绿色 10%
██  其他 5%
```

**设计理由**:
- 传统扑克牌桌的经典配色
- 沉稳不刺眼，长时间观看不疲劳
- 与白色牌面形成高对比度 (8.5:1)

---

### 2. 简洁不花哨

**执行标准**:
- ❌ 无渐变
- ❌ 无阴影
- ❌ 无多余装饰
- ✅ 扁平化设计
- ✅ 功能优先

**对比**:
```
❌ 错误示例:  [按钮] (带渐变 + 阴影 + 高光)
✅ 正确示例:  [按钮] (纯色 + 边框)
```

---

### 3. 手机友好

**关键指标**:
| 指标 | 目标值 | 实际值 | 状态 |
|------|--------|--------|------|
| 最小点击区 | ≥44px | 44px | ✅ |
| 手牌高度 | ≥150px | 180px | ✅ |
| 最小字号 | ≥11px | 20px | ✅ |
| 对比度 | ≥4.5:1 | 8.5:1 | ✅ |

**主色调**:

**辅助色**:
```css

```

**无障碍检查**:
- ✅ 深绿/白色对比度：8.5:1 (AA)
- ✅ 白色/黑色对比度：21:1 (AAA)
- ✅ 所有组合通过 WCAG AA 标准

---

### 字体规范

**字号层级** (不超过 5 级):
| 层级 | 字号 | 用途 |
|------|------|------|

| 大按钮 | 32px | 主按钮文字 |

| 辅助 | 24px | 说明文字 |
| 小字 | 20px | 注释、提示 |

**字体系列**:

```css
                   "Helvetica Neue", STHeiti, sans-serif;
```
- 使用系统默认字体
- 无需外部加载

```
主按钮 → 次要按钮：24px
手牌 → 操作按钮：20px
```

**水平间距**:

## 交互设计

### 选牌交互

```
点击牌 → 向上偏移 20px → 黄色高亮
        (动画 0.2s)
```

### 出牌交互

```
选牌 → 点击"出牌" → 验证
                  ├─ 成功 → 飞入出牌区
                  └─ 失败 → 震动 + 红色提示
```

### 暂停交互

```
点击暂停 → 半透明遮罩 → 继续/退出
          (rgba(0,0,0,0.6))
```

## 关键设计决策

### 决策 1: 深绿色主色调

**选项对比**:
| 方案 | 优点 | 缺点 | 选择 |
|------|------|------|------|
| 深绿色 (#1a472a) | 传统、沉稳、护眼 | 较暗 | ✅ |
| 浅绿色 (#4caf50) | 清新、明亮 | 刺眼、不传统 | ❌ |
| 蓝色系 | 现代、科技感 | 不符合扑克传统 | ❌ |

**决策理由**: 掼蛋是传统扑克游戏，深绿色牌桌是经典配色，符合用户心智模型。

---

### 决策 2: 系统字体优先

**选项对比**:
| 方案 | 优点 | 缺点 | 选择 |
|------|------|------|------|
| 系统字体 | 无需加载、跨平台一致 | 个性化弱 | ✅ |
| 自定义字体 | 个性化强、品牌感 | 加载慢、包体大 | ❌ |

**决策理由**: 掼蛋是轻量级游戏，启动速度优先，系统字体完全满足需求。

---

### 决策 3: 扁平化设计

**选项对比**:
| 方案 | 优点 | 缺点 | 选择 |
|------|------|------|------|
| 扁平化 | 简洁、性能好、易维护 | 视觉层次弱 | ✅ |
| 拟物化 | 视觉层次强、真实感 | 复杂、过时 | ❌ |

**决策理由**: 符合"简洁、不花哨"原则，减少视觉干扰，让用户专注牌局。

---

### 决策 4: 手牌 180px 高度

**选项对比**:
| 方案 | 优点 | 缺点 | 选择 |
|------|------|------|------|
| 150px (最低要求) | 可见牌数多 | 识别度低 | ❌ |
| 180px (实际选择) | 清晰易读、有余量 | 可见牌数略少 | ✅ |
| 200px+ | 非常清晰 | 占用空间大 | ❌ |

**决策理由**: 手机屏幕小，牌面清晰度第一，180px 在清晰度和可见牌数间取得平衡。
