  ↓ 次要按钮 (60px)
  ↓ 100px

  ↓ 80px

  ↓ 44px
```

```
左边缘
  ↓ 32px
按钮左边缘 (按钮宽 280px，居中)
  ↓ 280px
按钮右边缘
  ↓ 32px
右边缘
```

**居中计算**:
```
(750px - 280px) / 2 = 235px (按钮左边缘距屏幕左侧)
