## 🃏 卡牌设计规范（v2.0 修正版）

### 花字布局（关键修正）

```
┌────────────────┐
│ A              │ ← 左上角：字 + 花（竖排）
│ ♥              │
│                │
│                │
│         ♥      │ ← 中间：大花色（偏右下）
│                │
│                │
│                │ ← 右下角：完全留空（无任何内容）✅
└────────────────┘
```

### 布局要点
- ✅ **左上角**: 字 + 花竖排（字在上 28px，花在下 20px）
- ✅ **中间**: 花色 48px，中心偏右下（向右偏移 8px，向下偏移 4px）
- ❌ **右下角**: **完全留空，无任何花字**（v2.0 关键修正）

### 卡牌尺寸
- **宽度**: 80px
- **高度**: 112px
- **比例**: 1:1.4
- **圆角**: 8px

## 🎨 区域设计规范

### 1. 顶部区域（对家信息）
- **位置**: 上部中央 (960, 80)
- **尺寸**: 150×100px
- **内容**:
  - 头像（圆形，60×60px）
  - 玩家昵称（16px 白色）
  - 剩余牌数（20px 黄色高亮）

### 2. 左侧区域（上家信息）
- **位置**: 左部中央 (80, 540)
- **尺寸**: 150×100px
- **内容**: 同对家

### 3. 右侧区域（下家信息）
- **位置**: 右部中央 (1840, 540)
- **尺寸**: 150×100px
- **内容**: 同对家

### 4. 中央区域（游戏区）
- **牌堆**: 左侧 (300, 450)，卡牌背面图（80×112px）
- **出牌区**: 中央 (700, 350)-(1220, 730)，圆角矩形框
- **最后出牌提示**: 出牌区上方，半透明黑色背景 + 黄色文字

### 5. 底部区域（玩家手牌）
- **手牌区**: 底部中央 (960, 850)
- **卡牌排列**: 横向扇形展开，间距 20px 重叠
- **悬停效果**: 卡牌上移 20px，放大 1.1 倍

### 6. 操作按钮区
```
┌─────────────────────────────────┐
│   [💡 提示]  [✅ 出牌]  [❌ 不出]   │
└─────────────────────────────────┘
```
- **位置**: 手牌下方 (960, 990)
- **按钮尺寸**: 120×40px
- **间距**: 16px
- **样式**: 圆角矩形，渐变背景
  - 提示: 蓝色 (#1976d2)
  - 出牌: 绿色 (#388e3c)
  - 不出: 红色 (#d32f2f)
