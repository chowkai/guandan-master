### 6. 游戏页

```
┌─────────────────────────────┐
│ ⏰ 15s    李四的回合         │
├─────────────────────────────┤
│                             │
│    张三 (15 张)  王五 (15 张)  │
│                             │
│        ┌───────┐            │
│        │ 出牌区 │            │
│        │ 3333  │            │
│        └───────┘            │
│                             │
│  ┌─────────────────────┐   │
│  │ 🂡🂢🂣🂤🂥... (手牌) │   │
│  └─────────────────────┘   │
│                             │
│ [提示] [不出] [出牌] [托管] │
│                             │
└─────────────────────────────┘
```

**交互说明**:
- 点击手牌 → 选牌/取消选牌
- 滑动牌 → 理牌
- 点击"提示" → 推荐可出的牌
- 点击"不出" → 跳过本轮
- 点击"出牌" → 出牌
- 点击"托管" → 自动出牌

## 🎨 设计规范

```
主色调: #1890ff (蓝色)
辅助色: #52c41a (绿色 - 胜利)
警告色: #faad14 (黄色)
错误色: #f5222d (红色)

背景色: #f5f5f5
卡片色: #ffffff
文字色: #333333 / #666666 / #999999
```

### 字体系统
```
标题：32px / 600
正文：28px / 400
辅助文字：24px / 400
按钮：32px / 500
```

### 间距系统
```
4px, 8px, 12px, 16px, 24px, 32px
```

```

卡片：12px
```
