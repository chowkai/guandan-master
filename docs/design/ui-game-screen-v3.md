# 游戏界面 UI 设计规范 v3.0

> 更新时间：2026-03-23  
> 版本：v3.0（基于卡牌设计 v2.0 修正）

---

## 📱 界面概览

### 屏幕规格
- **分辨率**: 1920×1080 (16:9)
- **安全区域**: 上下左右各留 40px 边距
- **背景**: 深绿色渐变 (#1a472a → #2d5a3f)

---

## 🎴 卡牌布局（v2.0 修正版）

### 玩家手牌区
```
┌─────────────────────────────────────────────────────┐
│                                                     │
│                    对手区域                          │
│                                                     │
│              [牌堆]        [出牌区]                  │
│                                                     │
├─────────────────────────────────────────────────────┤
│                                                     │
│         [手牌 1] [手牌 2] [手牌 3] ... [手牌 N]      │
│                                                     │
│                    操作按钮区                        │
│              [提示] [出牌] [不出]                    │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 手牌排列
- **卡牌尺寸**: 80×112px（1:1.4 比例）
- **间距**: 卡牌之间重叠 20px（视觉紧凑）
- **扇形展开**: 可选，角度 15°
- **悬停效果**: 卡牌上移 20px，放大 1.1 倍

---

## 🎨 区域设计规范

### 1. 顶部区域（对手信息）
- **高度**: 120px
- **背景**: 半透明黑色 (#00000080)
- **内容**:
  - 对手头像（左侧，60×60px）
  - 对手昵称（头像右侧，20px 白色）
  - 剩余牌数（右侧，24px 黄色）

### 2. 中央区域（游戏区）
- **高度**: 400px
- **内容**:
  - **牌堆**: 左侧，卡牌背面图（80×112px）
  - **出牌区**: 中央，已出卡牌展示区
  - **最后出牌**: 出牌区上方显示最后一家出的牌

### 3. 底部区域（玩家手牌）
- **高度**: 200px
- **背景**: 半透明黑色渐变 (#00000080 → #00000000)
- **手牌区**: 中央横向排列
- **操作按钮区**: 手牌下方

### 4. 操作按钮区
```
┌─────────────────────────────────┐
│   [💡 提示]  [✅ 出牌]  [❌ 不出]   │
└─────────────────────────────────┘
```
- **按钮尺寸**: 120×40px
- **间距**: 16px
- **样式**: 圆角矩形，渐变背景
- **提示**: 蓝色 (#1976d2)
- **出牌**: 绿色 (#388e3c)
- **不出**: 红色 (#d32f2f)

---

## 🃏 卡牌设计规范（v2.0）

### 花字布局
- **左上角**: 字 + 花竖排（字在上，花在下）
- **中间**: 花色偏右下（给左上角留空间）
- **右下角**: 花 + 字竖排（倒置）

### Joker 设计
- **小王**: 蓝色渐变背景 + 👤图标 + "小王"文字
- **大王**: 红色渐变背景 + 👑图标 + "大王"文字

详见：`card-design-spec-v2.md`

---

## 🎭 动画效果

### 出牌动画
- **持续时间**: 300ms
- **缓动**: ease-out
- **效果**: 手牌飞向出牌区，缩放至 0.8 倍

### 胜利动画
- **持续时间**: 2000ms
- **效果**: 彩带飘落 + 卡牌翻转展示
