```

### 手牌区详细说明

```
┌─────────────────────────────────┐
│  玩家 4 (你)                     │
│                                 │
│  🂡 🂢 🂣 🂤 🂥 🂦 🂧 🂨 🂩 🂪 🂫  │
│  (150px 高度，白色边框 3px)       │
│                                 │
│  [不出] [提示] [排序▼] [出牌]    │
│   (灰色) (禁用)  (下拉)  (蓝色)  │
└─────────────────────────────────┘
```

### 交互说明

1. **选牌**: 单击选中（高亮）
2. **出牌**: 选牌后点击"出牌"
3. **不出**: 直接点击"不出"
4. **排序**: 点击展开排序选项
   - 按大小
   - 按花色
   - 按类型

### 设计要点

- **牌面大**: 150px 高度，清晰可见
- **操作少**: 常用操作 1 次点击
- **无干扰**: 无花哨动画

```
┌─────────────────────────────────┐

│      游戏已暂停                  │
│                                 │
│    ┌─────────────────────┐      │
│    │                     │      │
│    │      继续游戏        │      │
│    │                     │      │
│    └─────────────────────┘      │
│                                 │
│    ┌─────────────────────┐      │
│    │                     │      │

│    │                     │      │

### 设计要点

- **防止误触**: 退出需要确认
- **快速继续**: 继续游戏 1 次点击

## 🎨 设计规范

### 色彩

| 用途 | 颜色 | 说明 |
|------|------|------|
| 牌桌背景 | #1a472a | 深绿色，护眼 |
| 手牌背景 | #ffffff | 白色 |
| 手牌边框 | #ffffff | 白色 3px |
| 主按钮 | #4CAF50 | 绿色，行动色 |
| 次按钮 | #f5f5f5 | 灰色，次要色 |
| 文字 | #333333 | 深灰色 |

### 字体

- **标题**: 24px，加粗
- **正文**: 16px，常规
- **按钮**: 18px，加粗

### 间距

- **大间距**: 24px
- **中间距**: 16px
- **小间距**: 8px
