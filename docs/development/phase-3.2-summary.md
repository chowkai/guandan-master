## 使用说明

### 在微信开发者工具中运行

1. 打开微信开发者工具
2. 导入项目：选择 `guandan-miniprogram` 目录
3. 填写 AppID (测试可选"测试号")
4. 点击"编译"即可预览

### 真机测试

1. 点击"预览"生成二维码
2. 微信扫码在手机上查看
3. 可在开发者工具查看手机日志
