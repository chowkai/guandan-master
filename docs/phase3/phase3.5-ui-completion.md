## 🔧 技术细节

### 页面路由配置

**app.json**:
```json
{
  "pages": [
    "pages/index/index",
    "pages/game/game",
    "pages/settings/settings",
    "pages/rules/rules"
  ],
  ...
}
```

### 样式规范

- 使用 CSS 变量统一管理颜色和间距
- 采用 rpx 单位实现响应式布局
- 遵循微信小程序设计规范

### 交互设计

- 按钮点击热区 ≥ 44px（符合微信规范）
- 按下动画流畅（transition 0.2s）
- Toast 提示反馈（1.5 秒）
- Modal 确认对话框（重置设置）
