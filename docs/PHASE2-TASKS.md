**负责人**: design-agent (产品经理角色)  
**优先级**: P0  
**预计耗时**: 30 分钟 (基准 5 分钟 × 复杂度 2x + 50% Buffer)

**工作内容**:

4. 设计交互流程（开始游戏/暂停/继续/结束）

**交付物**:
- [ ] `docs/design/prototype-main-screen.md` - 主界面原型

- [ ] `docs/design/prototype-summary.md` - 原型汇总文档

**验收标准**:
- [ ] 主界面 3 秒内理解（开始游戏按钮突出）
- [ ] 游戏界面牌面清晰（手牌≥150px）
