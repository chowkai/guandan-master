# 域名备案指南

本文档描述在中国大陆服务器上部署网站时所需的域名备案流程。

## 什么是域名备案？

根据中国法律法规，在中国大陆境内的服务器上提供网站服务，需要对域名进行 ICP 备案（Internet Content Provider 备案）。

## 备案前提条件

- ✅ 中国大陆境内的服务器（如腾讯云服务器）
- ✅ 已注册的域名
- ✅ 实名认证的域名
- ✅ 有效的身份证件（个人/企业）

## 备案所需材料

### 个人备案

| 材料 | 要求 | 示例 |
|------|------|------|
| 身份证 | 正反两面彩色扫描件 | JPG/PNG, 清晰可辨 |
| 域名证书 | 域名注册商处获取 | PDF 格式 |
| 人脸识别 | 备案过程中完成 | 手机小程序扫码 |
| 联系方式 | 手机号 + 邮箱 | 需能接收验证码 |
| 网站名称 | 符合规范 | 不能含「中国」「国家」等字样 |

### 企业备案

| 材料 | 要求 | 示例 |
|------|------|------|
| 营业执照 | 彩色扫描件 | 三证合一，JPG/PNG |
| 法人身份证 | 正反两面彩色扫描件 | JPG/PNG |
| 经办人身份证 | 正反两面彩色扫描件（如适用） | JPG/PNG |
| 域名证书 | 域名注册商处获取 | PDF 格式 |
| 授权书 | 如经办人非法人 | 需盖公章 |
| 公章 | 电子签章 | 备案文件中需盖章 |

## 备案流程步骤

### 第一步：准备阶段（1-2 天）

1. **域名实名认证**
   - 登录域名注册商后台
   - 完成域名实名认证
   - 确保域名信息与实际备案主体一致

2. **获取域名证书**
   - 在域名注册商处下载域名证书
   - 保存为 PDF 格式备用

3. **准备网站信息**
   - 网站名称（个人建议：XXX 的个人空间）
   - 网站类别（个人博客/企业官网等）
   - 网站服务内容描述

### 第二步：提交备案（腾讯云）

1. **登录备案系统**
   ```
   https://beian.cloud.tencent.com/
   ```

2. **填写备案信息**
   - 主体信息（个人/企业基本信息）
   - 网站信息（域名、名称、类别）
   - 服务器信息（选择对应的云服务器）

3. **上传材料**
   - 按系统要求上传所有材料
   - 确保图片清晰、完整
   - 文件格式符合要求

4. **人脸识别**
   - 使用微信小程序扫码
   - 完成人脸核验
   - 签署备案承诺书

5. **提交初审**
   - 检查所有信息无误
   - 提交至腾讯云初审

### 第三步：腾讯云初审（1-2 工作日）

- 腾讯云客服审核材料完整性
- 如有问题，会短信/电话通知补充
- 初审通过后提交至通信管理局

### 第四步：通信管理局审核（10-20 工作日）

- 各省通信管理局审核
- 审核期间保持电话畅通
- 可能需要配合电话核验

### 第五步：备案完成

- 审核通过后，获得 ICP 备案号
- 格式：省份 ICP 备 XXXXXXXX 号
- 需在网站首页底部展示备案号

## 备案时间线

```
准备材料 (1-2 天)
    ↓
腾讯云初审 (1-2 工作日)
    ↓
通信管局审核 (10-20 工作日)
    ↓
备案完成 ✅
```

**总耗时**: 约 15-25 个工作日（3-5 周）

## 备案注意事项

### ⚠️ 常见驳回原因

1. **材料问题**
   - 图片模糊、不完整
   - 身份证过期
   - 域名未实名认证

2. **信息问题**
   - 主体信息不一致
   - 网站名称违规
   - 联系方式错误

3. **网站内容问题**
   - 涉及敏感内容
   - 个人备案含商业信息
   - 网站名称夸大其词

### ✅ 提高通过率建议

1. **材料准备**
   - 使用高清扫描件
   - 确保在有效期内
   - 信息完全一致

2. **网站命名**
   - 个人：XXX 的个人空间/博客
   - 企业：公司简称 + 官网
   - 避免：中国、国家、国际等字样

3. **网站内容**
   - 备案期间网站不能访问
   - 备案完成后先上线简单内容
   - 避免敏感、违规内容

4. **联系方式**
   - 保持手机畅通
   - 及时查看短信/邮件
   - 配合核验电话

## 备案后事项

### 1. 展示备案号

在网站首页底部添加：

```html
<footer>
  <p>
    <a href="https://beian.miit.gov.cn/" target="_blank">
      京 ICP 备 XXXXXXXX 号
    </a>
  </p>
</footer>
```

### 2. 公安备案（可选）

部分省市要求公安备案：
- 网址：http://www.beian.gov.cn/
- 时限：备案成功后 30 日内
- 展示：网站首页添加公安备案号

### 3. 备案变更

如信息变更，需及时更新备案：
- 主体信息变更
- 网站信息变更
- 服务器变更

### 4. 备案注销

如不再使用，需注销备案：
- 登录腾讯云备案系统
- 申请注销备案
- 避免影响其他备案

## 常见问题 FAQ

### Q1: 个人备案可以用于商业用途吗？

**A**: 不可以。个人备案仅限个人非商业用途。如需商业运营，需办理企业备案。

### Q2: 备案期间网站可以访问吗？

**A**: 不可以。备案期间服务器不能提供 Web 服务，否则会被驳回。

### Q3: 一个主体可以备案多个网站吗？

**A**: 可以。一个主体（个人/企业）可以备案多个域名，但需分别提交。

### Q4: 备案成功后可以更换服务器吗？

**A**: 可以。但需要办理「备案接入」，将备案信息接入新的服务商。

### Q5: 备案被驳回怎么办？

**A**: 
1. 查看驳回原因
2. 修改问题内容
3. 重新提交
4. 如有疑问，联系腾讯云客服

### Q6: 域名过期会影响备案吗？

**A**: 会。域名过期可能导致备案被注销，需及时续费。

### Q7: 备案需要付费吗？

**A**: 备案本身免费。但需要：
- 域名费用（约 50-100 元/年）
- 服务器费用（按实际配置）
