# 代码提交检查清单 (固化版)

**适用范围**: 所有项目  
**版本**: v1.0  
**生效日期**: 2026-03-21

---

## ⚠️ 提交前必须执行 (禁止跳过)

### 1. 编译检查 (必须通过)

```bash
npm run build
```

**检查项**:
- [ ] 编译成功
- [ ] 无错误
- [ ] 无警告

**失败处理**: 修复所有错误后重新编译

---

### 2. TypeScript 检查 (必须通过)

```bash
# 如有 TypeScript
npx tsc --noEmit
```

**检查项**:
- [ ] 无类型错误
- [ ] 无语法错误
- [ ] 无隐式 any
- [ ] 无未使用变量

**失败处理**: 修复所有错误

---

### 3. 启动测试 (必须通过)

```bash
npm run dev
# 访问页面，检查是否正常运行
```

**检查项**:
- [ ] 服务器正常启动
- [ ] 页面无白屏
- [ ] Console 无错误
- [ ] 核心功能可用

**失败处理**: 修复后重新测试

---

### 4. 功能测试 (必须通过)

**核心功能清单**:
- [ ] 开新局功能
- [ ] 发牌功能
- [ ] 选牌功能
- [ ] 出牌功能
- [ ] AI 出牌功能
- [ ] 提示条功能

**失败处理**: 修复后重新测试

---

### 5. 性能检查 (必须达标)

**检查项**:
- [ ] 加载时间 < 2 秒
- [ ] 帧率 ≥ 60 FPS
- [ ] 无明显卡顿

**失败处理**: 优化后重新测试

---

### 6. 代码规范 (必须符合)

**检查项**:
- [ ] 缩进一致 (2 空格)
- [ ] 命名规范
- [ ] 无 console.log (生产代码)
- [ ] 有必要的注释
- [ ] 函数职责单一
- [ ] 无重复代码

**失败处理**: 重构后重新审查

---
