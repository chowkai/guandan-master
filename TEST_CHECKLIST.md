# 掼蛋大师 v1.2.1 - 测试验证清单

## 🎯 测试目标
验证开新局随机出牌和出牌提示功能是否正常

## 📋 测试步骤

### 测试 1: 新游戏随机先手
1. 打开游戏页面
2. 点击"🎮 新游戏"按钮
3. 观察顶部提示条

**预期结果**:
- [ ] 提示条显示"XX 先出牌"（随机一个玩家）
- [ ] 如果是"我"先出牌，提示条为绿色
- [ ] 如果是 AI 先出牌，提示条为红色
- [ ] 底部状态栏显示相同信息
- [ ] 可以正常出牌（如果是玩家回合）

### 测试 2: 出牌提示切换
1. 开始新游戏
2. 等待轮流出牌
3. 观察提示条变化

**预期结果**:
- [ ] 每个玩家回合，提示条更新为对应玩家
- [ ] 玩家回合：绿色背景，"轮到你出牌"
- [ ] AI 回合：红色背景，"轮到 AI 左/右/对家 出牌"
- [ ] 提示条有脉冲动画效果

### 测试 3: 贡牌后先手
1. 完成一局游戏
2. 点击"再来一局"
3. 进入贡牌阶段（如果需要）
4. 贡牌结束后观察

**预期结果**:
- [ ] 单贡：末游先出牌
- [ ] 双贡：三游先出牌
- [ ] 抗贡：头游先出牌
- [ ] 提示条正确显示先手玩家

### 测试 4: 响应式显示
1. 在不同设备/窗口大小下测试
2. 平板尺寸（1024px）
3. 手机尺寸（768px, 480px）

**预期结果**:
- [ ] 提示条始终在顶部可见
- [ ] 字体大小适配屏幕
- [ ] 不遮挡游戏内容
