### 2. 测试开局流程 (P0)

**测试步骤**:
1. 点击 "🎮 新游戏" 按钮
2. **观察**: 发牌动画开始（卡牌逐张出现）
3. **等待**: 约 5 秒，直到 27 张牌全部发出
4. **检查**: 
   - ✅ 手牌是否正确显示 27 张
   - ✅ 顶部提示条显示 "轮到你出牌" 或 "XX 先出牌"
   - ✅ 底部状态栏显示当前出牌玩家
   - ✅ 出牌按钮可以点击

**预期结果**:
- 发牌动画流畅，卡牌逐张出现
- 发牌完成后，手牌自动排序
- 如果是玩家先手，出牌按钮可用
- 点击卡牌可以选中（卡牌上移）
- 点击"出牌"按钮可以正常出牌

**常见问题**:
- ❌ 发牌完成前就能点击出牌 → 等待发牌完成
- ❌ 手牌显示不全 → 检查浏览器控制台错误
- ❌ 出牌按钮灰色 → 不是玩家回合，等待 AI 出牌

### 3. 测试牌面设计 (P0)

**检查点**:
1. **字体大小**: 中央牌字应该是 48px，清晰可见
2. **花色大小**: 中央花色图标应该是 40px
3. **角标**: 左上角和右下角有小字角标
4. **颜色**: 
   - 红桃/方块 → 红色
   - 黑桃/草花 → 绿色
5. **大小王**:
   - 显示 "JOKER" 竖排文字
   - 显示 👑 王冠图标
   - 大王红色，小王绿色

**对比测试**:
- 截取一张牌，放大查看是否清晰
- 在手机上查看是否仍然清晰

### 4. 测试桌面布局 (P0)

**桌面端测试**:
1. 检查桌面是否固定尺寸（不随发牌扩展）
2. 检查四个玩家位置：
   - 上方：对家（AI）
   - 左方：AI 左
   - 右方：AI 右
   - 下方：我（玩家）
3. 检查手牌是否两行显示
4. 检查中央出牌区域

**手机端测试** (使用浏览器开发者工具):
1. 切换到手机模式（375x667）
2. 检查是否隐藏左右 AI 玩家
3. 检查手牌是否正常换行
4. 检查按钮是否正常显示

### 5. 测试出牌流程

**测试步骤**:
1. 等待轮到玩家出牌
2. 点击一张或多张牌（应该上移选中）
3. 点击"出牌"按钮
4. **检查**:
   - ✅ 牌从手牌中移除
   - ✅ 牌显示在出牌区域
   - ✅ 轮到下一个玩家
   - ✅ AI 自动出牌

## 🐛 已知问题排查

### 问题：发牌后无法出牌

**可能原因**:
1. 发牌未完成就点击出牌
2. 不是玩家回合（等待 AI）
3. 未选中牌

**解决方法**:
1. 等待发牌动画完全结束
2. 查看顶部提示条，确认轮到谁
3. 点击卡牌选中（应该上移）

### 问题：牌面显示异常

**可能原因**:
1. CSS 未正确加载
2. 浏览器缓存旧版本

**解决方法**:
1. 强制刷新：Ctrl+F5 或 Cmd+Shift+R
2. 清除浏览器缓存
3. 检查浏览器控制台错误

### 问题：手机端布局错乱

**可能原因**:
1. 浏览器不支持 CSS Grid
2. 视口设置问题

**解决方法**:
1. 使用现代浏览器（Chrome/Firefox/Safari）
2. 检查 meta viewport 设置
3. 尝试横屏模式
