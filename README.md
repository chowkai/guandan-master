## 🚀 快速开始

### 本地测试

```bash
# 1. 克隆仓库
git clone https://github.com/chowkai/guandan-master.git

# 2. 安装依赖
cd guandan-master
npm install

# 3. 运行测试
npm test

# 4. 启动信令服务器 (P2P 联机)
cd signaling-server
npm install
npm start
```

### 微信开发者工具

1. 打开微信开发者工具
2. 导入项目：`guandan-master/miniprogram`
3. 使用测试号 AppID
4. 点击"编译"运行
