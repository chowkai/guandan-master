## 🐛 已修复的核心 BUG

### BUG 1: 开新局无法出牌 ✅ 已修复

**v1 问题**: 
- DOM 元素为 null 导致 JS 错误
- currentTurn 未设置
- 提示条不更新

**v2 修复**:
```typescript
// Svelte 响应式 - 无需手动操作 DOM
game.setFirstPlayer()
// UI 自动更新！提示条自动变色！
```

### BUG 2: 发牌时 UI 卡顿 ✅ 已修复

**v1 问题**: 发牌动画阻塞主线程

**v2 修复**:
```typescript
// 异步发牌 + 响应式更新
setTimeout(() => {
  game.setFirstPlayer()
}, 3000) // 3 秒发牌时间
```

### BUG 3: 提示条颜色不变 ✅ 已修复

**v1 问题**: 状态与 UI 不同步

**v2 修复**:
```svelte
<!-- 响应式计算 -->
$: isPlayerTurn = $game.currentTurn === 'bottom'
$: bgColor = isPlayerTurn ? 'green' : 'red'
```

## 🎮 操作说明

### 鼠标操作
- **点击卡牌**: 选中/取消选中
- **点击"出牌"**: 打出选中的牌
- **点击"不要"**: 跳过本回合
- **点击"排序"**: 整理手牌

### 键盘快捷键
- **空格**: 出牌
- **ESC**: 取消选中
- **S**: 排序手牌
