## 🕵️ 斥候自主系统（更正版）

### 核心设计

**斥候和运输兵一样，完全自主工作**：
- ✅ **不需要玩家微操**（自动侦察）
- ✅ **配置后自动执行**（部署/驻防时自动侦察）
- ✅ **侦察细致度 = 配置的侦察兵力**
- ✅ **斥候遭遇战自动处理**

---

### 自主侦察机制

**部队配置侦察兵力后**：
```
玩家决策:
  "第 X 师前往 A 地区驻防"
  （该师已配置侦察连×1）

AI 自动执行:
  1. 运输连自动安排车辆
  2. 侦察连自动前出侦察
     - 自动分散搜索周边区域
     - 自动发现敌军/建筑/道路
     - 自动回报总部
  3. 通讯连自动保障通讯
     - 自动架设电台/电话线
     - 自动传递情报

玩家看到的结果:
  "第 X 师已到达 A 地区
   周边侦察完成
   发现：敌军×2 处，村庄×3，道路×2"
```

---

### 侦察细致度

**侦察效果 = 配置的侦察兵力**：

| 配置 | 侦察范围 | 侦察精度 | 更新频率 |
|------|---------|---------|---------|
| **无侦察兵力** | 仅部队视野 | 低 | 被动发现 |
| **侦察排** | 周边 5km | 中 | 每 30 分钟 |
| **侦察连** | 周边 10km | 高 | 每 15 分钟 |
| **侦察营** | 周边 20km | 很高 | 每 5 分钟 |

**玩家决策**：
```
配置多 = 情报准确及时，但占用战斗兵力
配置少 = 节省战斗兵力，但情报可能不完整
不配置 = 只能被动发现（容易被伏击）
```

---

### 斥候遭遇战

**双方斥候在野外遭遇**：
```
遭遇场景:
  我方：侦察连×1（100 人）
  敌方：侦察排×1（30 人）

自动处理:
  1. 战力对比
     - 人数：我方优势
     - 装备：自动计算
     - 地形：自动计算
  
  2. 战术选择（AI 自主）
     - 我方：追击/歼灭
     - 敌方：逃跑/隐蔽
  
  3. 战斗结果
     - 我方胜利：俘获敌方斥候
     - 敌方逃跑：部分逃脱回报
     - 两败俱伤：双方损失
  
  4. 情报影响
     - 我方胜利：继续侦察，掌握敌情
     - 敌方逃跑：敌方得知我军位置
     - 两败俱伤：双方侦察中断
```

**影响因素**：
| 因素 | 影响 |
|------|------|
| **人数优势** | 遭遇战胜负关键 |
| **装备优势** | 武器/车辆/通讯 |
| **地形优势** | 埋伏/隐蔽/逃跑 |
| **单兵能力** | 老兵/新兵/精英 |
| **战术指令** | 遭遇时优先任务（侦察/战斗/逃跑） |

---

### 斥候战术配置

**玩家可以预设战术优先级**：

| 战术模式 | 优先任务 | 遭遇敌人时 |
|---------|---------|-----------|
| **隐蔽侦察** | 发现敌情 | 优先隐蔽/逃跑 |
| **积极侦察** | 发现 + 驱赶 | 优势时战斗，劣势时逃跑 |
| **武装侦察** | 清除敌方斥候 | 优先战斗 |
| **快速侦察** | 快速覆盖 | 避免纠缠，继续侦察 |

**默认战术**：积极侦察（平衡型）

## 🎮 玩家体验流程

### 典型侦察作战（自主版）

```
1. 战前编制
   玩家决策:
   - 给第 X 师配置侦察连×1
   - 给第 X 师配置通讯连×1
   - 设定侦察战术：积极侦察
   
   （此后无需再管侦察细节）

2. 部署命令
   玩家下达:
   "第 X 师前往 A 地区驻防"
   
   （不需要单独命令侦察连）

3. AI 自动执行
   侦察连自动:
   - 自动前出 10km 侦察
   - 自动分散搜索
   - 自动发现目标
   - 自动回报总部
   
   运输连自动:
   - 自动安排车辆
   - 自动装载辎重
   
   通讯连自动:
   - 自动架设电台
   - 自动传递情报

4. 斥候遭遇战
   侦察连遭遇敌方斥候:
   
   AI 自动判断:
   - 我方 100 人 vs 敌方 30 人
   - 人数优势明显
   - 战术：积极侦察
   
   AI 自动决策:
   - 追击敌方斥候
   - 俘获/歼灭
   
   玩家看到:
   "侦察连报告：遭遇敌方斥候，已击溃，俘获×5"

5. 情报汇总
   玩家收到报告:
   "第 X 师侦察连报告:
    - A 地区发现敌军×2 处（约 1 个团）
    - 发现村庄×3（可征粮）
    - 发现道路×2（通往 B/C 地区）
    - 敌方斥候已被击溃
    - 情报更新时间：5 分钟前"

6. 战略决策
   玩家根据情报:
   - 决定进攻/防守/绕过
   - 下达新命令
   
   （侦察连继续自动侦察）
```

## 📊 设计意义

### 玩家价值

**玩家专注于**：
- ✅ 编制配置（配多少侦察兵）
- ✅ 战术预设（遭遇时如何处理）
- ✅ 战略决策（根据情报行动）

**AI 负责**：
- ✅ 自动侦察（分散/搜索/回报）
- ✅ 自动遭遇战（判断/战斗/追逃）
- ✅ 自动情报更新（持续侦察）

---

### 后勤系统统一

**所有后勤单位都自主工作**：
| 单位类型 | 自主工作 | 玩家决策 |
|---------|---------|---------|
| **运输连** | 自动运输 | 移动到哪里 |
| **侦察连** | 自动侦察 | 配置多少 |
| **通讯连** | 自动通讯 | 无需决策 |
| **维修连** | 自动维修 | 无需决策 |
| **医护连** | 自动救治 | 无需决策 |

**设计原则**：
```
玩家 = 战略决策者（做什么）
AI  = 战术执行者（怎么做）
```
