## 🎨 表现层系统

### 视觉风格

**确认**: **写实风格**

**设计原则**:
- ✅ 真实比例，真实色彩
- ✅ 符合历史背景的视觉呈现
- ✅ 不同时代装备/制服准确还原

**视角系统**:
| 视角层级 | 表现 | 说明 |
|---------|------|------|
| **战略层** | 地图标记 | 1km = 1 像素，显示编制符号 |
| **战役层** | 简化模型 | 100m = 1 像素，显示单位轮廓 |
| **战术层** | 精细模型 | 10m = 1 像素，可见个体 |
| **战斗层** | 细节渲染 | 1m = 1 像素，可见表情/装备细节 |

**分层渲染**:
- ✅ 地形层（最底层，不易改变）
- ✅ 地物层（道路、建筑、树木）
- ✅ 单位层（编制/个体）
- ✅ UI 层（信息叠加）

## ⚔️ 战斗系统

### 特殊战斗

**确认**: **夜战（含天气/温差等自然因素）**

**夜战机制**:
| 影响因素 | 影响内容 | 说明 |
|---------|---------|------|
| **视野** | 降低 | 可见距离减少 50-80% |
| **命中率** | 降低 | 远程武器命中率下降 |
| **偷袭成功率** | 提高 | 近战/伏击成功率上升 |
| **士气** | 波动 | 夜间恐慌度易上升 |
| **指挥效率** | 降低 | 传令兵效率下降 |

**天气系统**:
| 天气 | 影响 |
|------|------|
| **晴天** | 无影响 |
| **雨天** | 视野降低，火药受潮，移动力下降 |
| **雪天** | 视野降低，移动力大幅下降，寒冷度上升 |
| **雾天** | 视野大幅降低，伏击成功率提高 |
| **大风** | 远程武器命中率降低，火攻效果增强 |

**温差影响**:
```
寒冷度 = f(季节，时间，装备，天气)

影响:
- 寒冷度过高 → 体力消耗加快，士气下降
- 炎热度过高 → 口渴度加快，体力下降
- 个体需要相应装备（棉衣/防暑）
```

**其他自然因素**:
| 因素 | 影响 |
|------|------|
| **季节** | 春夏秋冬影响补给、移动、装备 |
| **地形** | 山地/平原/森林影响视野、移动、隐蔽 |
| **时间** | 昼夜循环影响视野、温度 |

### 巷战/水战/空战

**确认**: **暂不实现（后续 MOD 扩展）**

**首阶段聚焦**:
- ✅ 陆地常规战斗
- ✅ 夜战 + 天气系统
- ❌ 巷战（复杂度高）
- ❌ 水战（需要舰船系统）
- ❌ 空战（需要飞机系统）
- ❌ 攻城战（需要攻城器械）
- ❌ 游击战（需要特殊规则）

## 🎖️ 英雄单位系统

### 军官作用

**确认**: **军官影响团队到个人指标**

**军官影响范围**:
| 影响对象 | 影响指标 | 说明 |
|---------|---------|------|
| **编制整体** | 凝聚力、士气 | 军官存在提高阈值 |
| **编制整体** | 指挥效率 | 军官能力决定命令执行速度 |
| **个体** | 恐慌阈值 | 军官督战降低恐慌 |
| **个体** | 战斗效率 | 军官指挥提高命中率 |

**军官阵亡影响**:
```
军官阵亡 → 
1. 凝聚力立即下降
2. 指挥效率降低
3. 恐慌度上升
4. 可能触发群体崩溃

老兵可临时接管:
- 老兵自动接管指挥（如果有）
- 但效果不如正规军官
```

### 历史名将系统

**确认**: **历史名将在编辑器/MOD 中体现**

**获取方式**:
| 方式 | 说明 |
|------|------|
| **战役编辑器** | 官方 MOD 预设历史名将 |
| **沙盒模式** | 玩家手动添加历史名将 |
| **随机招募** | 能力很强的平民/士兵（运气） |

**名将能力**:
- ✅ 随机生成强力能力
- ✅ 能力类型多样（指挥/战术/后勤/鼓舞）
- ✅ 玩家需要培养与重用

**培养系统**:
```
成长路径:
新兵 → 老兵 → 军官 → 名将

培养方式:
- 实战经验积累
- 军功晋升
- 玩家重点培养（资源倾斜）
```

### 无武将单挑

**确认**: **不存在武将单挑情形**

**设计原则**:
- ❌ 无三国式武将单挑
- ❌ 无个人武力决定战局
- ✅ 强调集体作战、战略协调
- ✅ 名将作用在指挥而非个人战斗

## 🌍 地图与地形系统

### 地形分层

**确认**: **分层概念（技术验证后细化）**

**分层架构**:
```
Layer 0: 基础地形层（最底层，不易改变）
  - 高度数据（SRTM）
  - 基础地貌（山地/平原/盆地）
  
Layer 1: 地物层（上层，可改变）
  - 道路（可建造/破坏）
  - 建筑（可建造/占领/破坏）
  - 树木（可砍伐/燃烧）
  - 装备（可移动/部署）
  
Layer 2: 单位层
  - 编制单位
  - 个体单位
  
Layer 3: UI 层
  - 信息叠加
  - 命令标识
```

### 地图规模

**确认**: **无缝缩放（技术验证）**

**缩放层级**:
| 层级 | 比例尺 | 用途 |
|------|--------|------|
| **战略层** | 1km/像素 | 整体战局规划 |
| **战役层** | 100m/像素 | 编制调动 |
| **战术层** | 10m/像素 | 战斗指挥 |
| **战斗层** | 1m/像素 | 个体观察 |

**技术验证需求**:
- ✅ LOD 系统验证
- ✅ 无缝加载验证
- ✅ 内存管理验证

## 🏗️ 地图互动系统

### 地形改变

**确认**: **最底层地形不易改变，上层可互动**

**可改变对象**:
| 对象 | 可改变性 | 说明 |
|------|---------|------|
| **基础地形** | ❌ 不易改变 | 高度、地貌永久 |
| **道路** | ✅ 可建造/破坏 | 工兵建造/炮火破坏 |
| **建筑** | ✅ 可建造/占领/破坏 | 经济建筑/军事建筑 |
| **树木** | ✅ 可砍伐/燃烧 | 影响视野/隐蔽 |
| **装备** | ✅ 可移动/部署 | 火炮部署/障碍物 |

**互动类型**:
```
建造:
- 道路：工兵修建
- 建筑：经济/军事建筑
- 障碍物：壕沟/碉堡/路障

破坏:
- 炮火摧毁：建筑/道路/桥梁
- 工兵爆破：桥梁/建筑
- 燃烧：树木/建筑

占领:
- 建筑占领：获得控制权
- 区域占领：控制资源点
```

## 🎮 游戏模式系统

### 确认模式

**确认**: **5 种游戏模式**

| 模式 | 说明 | 状态 |
|------|------|------|
| **沙盒模式** | 自由设定参数，装备/战役编辑器游玩 | ✅ 确认 |
| **官方 MOD** | 红军战史、三国战史等官方战役 | ✅ 确认 |
| **征服者模式** | 从小到大的征服模式 | ✅ 确认 |
| **强军模式** | 基于强军模式的存档 | ✅ 确认 |
| **网络对决** | 基于强军模式存档的 PVP | ✅ 确认 |

### 模式详解

**沙盒模式**:
```
特点:
- 自由设定时代/地图/兵力
- 使用装备编辑器 DIY 装备
- 使用战役编辑器创作剧本
- 无胜利条件，自由游玩

用途:
- 测试装备/战术
- 创作 MOD
- 休闲娱乐
```

**官方 MOD**:
```
首发计划:
- 红军战史复刻
- 三国战史复刻

特点:
- 历史准确性
- 剧情引导
- 教学功能
```

**征服者模式**:
```
玩法:
- 从小势力开始
- 逐步征服扩大
- 管理资源/兵力/领土

胜利条件:
- 统一指定区域
- 达成特定目标
```

**强军模式**:
```
特点:
- 建设强大军队
- 完成特定挑战
- 可保存存档

用途:
- 网络对决基础
- 挑战模式
```

**网络对决**:
```
基础:
- 基于强军模式存档
- PVP 对战
- 排名系统

特点:
- 公平竞技
- 战术对抗
- 社交功能
```

## 📡 信息系统

### 通讯系统

**确认**: **通讯依赖传令兵，可能被切断拦截**

**通讯机制**:
| 通讯方式 | 距离 | 特点 |
|---------|------|------|
| **旗鼓号** | 视距内 | 快速简单命令 |
| **传令兵** | 任意距离 | 详细命令，可能被拦截 |
| **电台/电话** | 远距离 | 快速，需要装备 |

**通讯干扰**:
```
可能情况:
- ✅ 传令兵被拦截（命令延误/丢失）
- ✅ 电台被摧毁（通讯中断）
- ❌ 无假命令（无间谍系统）
- ❌ 无电子干扰（简化设计）
```

**设计原则**:
- ✅ 真实传递过程（需要时间）
- ✅ 风险存在（可能被拦截）
- ❌ 无复杂间谍系统
- ❌ 无假命令/策反

## 🤖 AI 系统

### NPC 个性系统

**确认**: **每个 NPC 都有个性、技能及各项指标**

**NPC 属性**:
```
基础属性:
- 年龄、性别、姓名
- 移动力、体力、负重力
- 护甲力、士气、恐慌度

技能系统:
- 武器技能（步枪/机枪/炮兵）
- 特殊技能（识途/护身/坚韧）
- 领导技能（军官专用）

个性特征:
- 勇敢/谨慎/激进
- 忠诚/自私/机会主义
- 经验值（新兵/老兵/精英）
```

### 难度设计

**确认**: **难度由 AI 能力指标决定**

**难度机制**:
```
难度 = AI 能力指标

影响对象:
- 己方 AI（友军）
- 敌方 AI（敌军）

调整方式:
- 简单：AI 决策慢，战术简单
- 普通：AI 正常水平
- 困难：AI 决策快，战术复杂
- 专家：AI 最优决策
```

**AI 能力指标**:
| 指标 | 说明 | 影响 |
|------|------|------|
| **决策速度** | AI 思考速度 | 反应时间 |
| **战术水平** | 战术选择质量 | 战斗效率 |
| **后勤管理** | 补给管理能力 | 持续作战能力 |
| **情报分析** | 情报利用能力 | 战略判断 |

## 📦 后勤与工作系统

### 自主工作

**确认**: **后勤/战斗兵根据笼统命令尽职工作**

**工作原则**:
```
玩家下达笼统命令 → 
编制/个体自主执行 → 
各尽其职完成工作

示例:
玩家："第 X 师驻防 A 地区"

自动执行:
- 战斗兵：占领阵地，警戒巡逻
- 后勤兵：建立补给线，运输物资
- 医护兵：设立医疗点，救治伤员
- 工匠：维修装备，建造工事
- 传令兵：传递命令，保持通讯
```

**设计哲学**:
- ✅ 玩家专注战略决策
- ✅ AI 负责战术执行
- ✅ 每个单位自动尽职
- ❌ 无需微操管理细节

## 🎵 音效系统

### 音效设计

**确认**: **近距离可听到战斗声音**

**音效层次**:
| 距离 | 音效内容 | 说明 |
|------|---------|------|
| **战斗层** (1m) | 枪炮声、喊杀声、呼吸声 | 细节音效 |
| **战术层** (10m) | 枪炮声、命令呼喊 | 战斗音效 |
| **战役层** (100m) | 沉闷炮火声 | 环境音效 |
| **战略层** (1km) | 无具体音效 | 背景音乐 |

**音效类型**:
```
环境音:
- 风声、雨声、鸟鸣
- 根据天气/时间变化

战斗音:
- 枪声、炮声、爆炸声
- 喊杀声、惨叫声
- 装备声（车辆/马匹）

语音:
- 军官命令
- 士兵呼喊
- 伤员求救

音乐:
- 背景音乐（不同时代/场景）
- 无版权风险
- 适合时代背景
```

### 音乐策略

**确认**: **找无版权风险的战争相关曲目**

**音乐来源**:
- ✅ 古典音乐（公共领域）
- ✅ 传统军乐（历史曲目）
- ✅ 原创音乐（委托创作）
- ✅ 开源音乐（Creative Commons）

**音乐使用**:
| 场景 | 音乐类型 |
|------|---------|
| **战前准备** | 紧张、期待 |
| **战斗进行** | 激烈、紧张 |
| **胜利** | 激昂、欢庆 |
| **失败** | 低沉、悲伤 |
| **战略地图** | 平静、思考 |
