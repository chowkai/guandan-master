## 🎯 核心设计简化

### 训练系统（简化）

**你的确认**：
- ✅ 训练只需要消耗**资金 + 时间**
- ✅ 个体 NPC**都可以学习多种技能**
- ❌ 不需要复杂的训练失败机制

**简化后设计**：
```
训练系统:
  消耗：资金 + 时间
  效果：提升熟练度（Lv1 → Lv5）
  限制：无（士兵可以学多种技能）
  
熟练度等级:
  Lv1 新兵 → Lv2 合格 → Lv3 熟练 → Lv4 老兵 → Lv5 精英
```

---

### 物理模拟（简化）

**你的确认**：
- ✅ 不要复杂计算
- ✅ 重点是**重量**和**补给消耗**

**简化后设计**：
```
核心属性:
  - 重量：影响士兵负重、移动力、体力消耗
  - 补给消耗：弹药/粮草/燃油的消耗速度
  
简化公式:
  体力消耗 = 武器质量 × 攻击次数 + 负重 × 移动距离
  
  伤害计算:
  基础伤害 = 武器攻击力 × (1 - 护甲防御/100)
  最终伤害 = 基础伤害 × 部位倍率
```

**移除复杂计算**：
- ❌ 不需要动能公式（质量×速度²）
- ❌ 不需要空气阻力
- ❌ 不需要尖锐度修正
- ✅ 保留重量和补给（核心）

```

    - 爆炸弹：爆炸范围 5m, 重量 0.05kg, 成本 10/发
```

### 平衡性限制

**防止过于破坏平衡**：
```
限制条件:
  1. 成本限制：过强的装备成本极高
  2. 重量限制：过重的装备无法携带
  3. 科技限制：某些装备需要编辑器解锁
  4. 生产限制：强大装备生产时间长
  5. 训练限制：需要相应技能才能使用
```

**示例：原子弹**
```
原子弹参数:
  重量：5000kg（需要战略轰炸机携带）
  射程：1000km（需要导弹投送）
  爆炸范围：5000m
  爆炸伤害：1000000（必杀）
  成本：10000000（极其昂贵）
  生产时间：365 天（1 年）
  使用限制：需要"核技术"解锁
  
实际影响:
  - 可以制造，但成本极高
  - 需要配套投送系统（导弹/轰炸机）
  - 生产周期长，无法量产
  - 使用会引发国际反应（如果有多国系统）
```
