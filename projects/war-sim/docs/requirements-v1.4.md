## 🏛️ 建筑系统（扩展）

### 核心原则

**建筑可以破坏，也可以占领**：
- ✅ 敌方可以摧毁经济建筑
- ✅ 敌方可以占领经济建筑
- ✅ 建筑内有平民工作人员

---

### 建筑占领机制

**占领流程**：
```
军队进入建筑范围
   ↓
驱逐/俘虏敌方守卫（如有）
   ↓
控制建筑
   ↓
接收建筑内平民
   ↓
决定是否继续运营
```

**占领后选择**：
| 选择 | 说明 | 效果 |
|------|------|------|
| **继续运营** | 保留平民，继续生产 | 获得产出，但需支付工资 |
| **拆除设备** | 拆走关键设备 | 获得资源，建筑报废 |
| **彻底摧毁** | 炸毁建筑 | 敌方无法使用，但失去产出 |
| **弃置** | 不管不顾 | 无收益，可能被敌方重新占领 |

---

### 平民系统（新增）

**经济建筑内的工作人员**：
- ✅ 大部分是**平民**（非士兵）
- ✅ 被俘后成为**敌方人员/工人**
- ✅ 个性随机生成（忠诚度/性格/经历）

**平民属性**：
| 属性 | 说明 | 取值范围 |
|------|------|---------|
| **姓名** | 随机生成 | 姓氏库 + 名字库 |
| **年龄** | 影响工作效率 | 16-60 岁 |
| **性别** | 影响岗位适配 | 男/女 |
| **技能** | 工作效率倍率 | Lv1-5 |
| **忠诚度** | 影响反抗/逃跑概率 | 0-100 |
| **性格** | 勇敢/懦弱/顺从/反抗 | 随机 |
| **从军经历** | 是否当过兵 | 是/否 |

---

### 平民行为逻辑

**被俘后反应**：
```
平民个性判定:
  - 忠诚度高 + 从军经历 → 可能逃跑/反抗
  - 忠诚度低 + 无经历 → 顺从工作
  - 性格勇敢 → 可能反抗
  - 性格懦弱 → 顺从或装病
```

**行为概率**：
| 行为 | 触发条件 | 概率 |
|------|---------|------|
| **顺从工作** | 忠诚度低 + 性格懦弱 | 70% |
| **消极怠工** | 忠诚度中 + 性格顺从 | 20% |
| **逃跑** | 忠诚度高 + 从军经历 | 5% |
| **反抗** | 忠诚度高 + 性格勇敢 + 从军经历 | 3% |
| **破坏设备** | 忠诚度极高 + 性格反抗 | 2% |

---

### 平民管理

**玩家决策点**：
| 决策 | 说明 | 效果 |
|------|------|------|
| **雇佣** | 支付工资，继续工作 | 获得产出，消耗资金 |
| **遣散** | 发放路费，让其回家 | 失去产出，节省资金 |
| **囚禁** | 关押为战俘 | 消耗粮食，防止逃跑 |
| **处决** | 杀死（不推荐） | 士气下降，民心下降 |

**工资系统**：
```
工资 = 基础工资 × 技能倍率 × 地区经济系数

例：
  基础工资：10 资金/天
  技能 Lv3: ×1.5
  发达地区：×1.2
  总工资：10 × 1.5 × 1.2 = 18 资金/天
```

## 💰 简化经济系统

### 核心原则

**简化但不失深度**：
- ❌ 不需要复杂的多资源系统
- ✅ 保留**人口**和**经济程度**概念
- ✅ 影响征兵/招工/税收

---

### 地域属性

每个地区（城市/乡村）有以下属性：

| 属性 | 说明 | 影响 |
|------|------|------|
| **人口** | 地区总人口 | 征兵上限、招工速度 |
| **经济程度** | 发达/普通/贫困 | 税收收入、工资水平 |
| **民心** | 对统治者的支持度 | 税收效率、反抗概率 |
| **治安** | 秩序稳定程度 | 生产安全、反抗概率 |

---

### 人口系统

**人口作用**：
```
1. 兵源
   可征兵数量 = 人口 × 征兵比例（通常 1-5%）
   
2. 劳动力
   招工速度 = 人口 × 经济程度系数
   
3. 税收基数
   税收 = 人口 × 人均税收 × 经济程度
```

**人口流动**：
```
战争影响:
  - 战败 → 人口减少（伤亡 + 逃亡）
  - 占领 → 人口变化（部分逃亡）
  - 长期驻军 → 人口稳定
  
经济影响:
  - 经济发达 → 人口流入
  - 经济贫困 → 人口流出
```

---

### 经济程度

**等级划分**：
| 等级 | 名称 | 税收系数 | 征兵意愿 | 招工速度 |
|------|------|---------|---------|---------|
| Lv1 | 贫困 | 0.5x | 高（为生计） | 慢 |
| Lv2 | 普通 | 1.0x | 中 | 中 |
| Lv3 | 发达 | 1.5x | 低（生活好） | 快 |
| Lv4 | 繁荣 | 2.0x | 很低 | 很快 |

**经济程度影响**：
```
税收 = 人口 × 基础税率 × 经济系数

例：
  人口 10 万，基础税率 10/人，经济发达（1.5x）
  税收 = 100000 × 10 × 1.5 = 150 万资金/月
```

---

### 税收系统

**简化公式**：
```
月收入 = 人口 × 人均税收 × 经济系数 × 民心系数

民心系数:
  - 民心 100-80%: 1.2x（踊跃纳税）
  - 民心 80-60%: 1.0x（正常）
  - 民心 60-40%: 0.8x（拖欠税款）
  - 民心 40-0%: 0.5x（抗税 + 反抗风险）
```

**支出项目**：
| 项目 | 说明 | 计算方式 |
|------|------|---------|
| **军饷** | 士兵工资 | 士兵数 × 单兵工资 |
| **工人工资** | 经济建筑工人 | 工人数 × 技能工资 |
| **维护费** | 建筑/装备维护 | 建筑价值 × 维护率 |
| **补给采购** | 粮草/弹药/燃油 | 消耗量 × 单价 |

**收支平衡**：
```
盈余 = 收入 - 支出

盈余 > 0: 资金积累，可扩大军队/建设
盈余 < 0: 资金消耗，可能欠饷（士气下降）
```

## 📊 简化后的经济系统总览

### 核心循环

```
人口 + 经济程度 → 税收收入
              ↓
         资金分配
              ↓
    军饷/工资/维护/补给
              ↓
         军队战斗力
              ↓
    占领更多地盘
              ↓
    更多人口 + 经济
```

### 玩家决策点

**需要考虑**：
- ✅ 征兵比例（影响生产力和税收）
- ✅ 税率高低（影响民心和收入）
- ✅ 工资水平（影响忠诚度和效率）
- ✅ 资金分配（军饷 vs 建设 vs 补给）

**不需要考虑**：
- ❌ 复杂资源管理（如每种弹药单独计算）
- ❌ 复杂贸易系统（如进出口）
- ❌ 复杂外交系统（如盟约/条约）

## 🎮 游戏设计意义

### 战略深度

**人口与经济**：
- 占领富庶地区 → 税收高但征兵难
- 占领贫困地区 → 税收低但兵源足
- 需要在收入和兵源间权衡

**平民管理**：
- 优待平民 → 忠诚度高，生产稳定
- 压迫平民 → 可能反抗，生产下降
- 需要在成本和稳定间权衡
