# 战争游戏 - 需求说明书 v1.2

> 基于个体微观决策的宏观战争模拟平台

**版本**: v1.2 (完整整合版)  
**创建日期**: 2026-03-20  
**状态**: 需求论证完成，准备技术验证

---

## 📋 核心理念

**战争是由无数微小个体的决策构成的**。每个士兵不是冷冰冰的数字，而是有生存本能、有归属感、有晋升欲望的"人"。

**游戏定位**：
- ✅ **战争模拟平台**（复现历史所有战争）
- ✅ **战略指挥游戏**（人性洞察 + 战略布局）
- ✅ **历史沙盒**（见证/改写历史）

**核心思想**：
- ✅ 每个单位都是独立个体，有自己的生命周期
- ✅ 个体行为遵循优先级原则（生存 > 编制 > 晋升）
- ✅ 玩家不能违背"人性"（不能随意牺牲个体）
- ✅ 宏观战争结局由微观个体决策汇聚而成

## 🎯 核心三问（已确认）

### 1️⃣ 乐趣点

**战略布局 + 人性洞察 + 历史沉浸感**
- 运筹帷幄之中，决胜千里之外的成就感
- 见证历史，改写历史的沉浸感
- 不是微操快感，不是数值碾压

### 2️⃣ 指挥边界

**玩家可以指挥任意层级编制**：
- 从方面军到班，都可以直接指挥
- 经济建筑可以移动（需要付出迁移代价）
- **不能**直接指挥个体士兵

**迁移代价**：
- 时间成本（拆卸 + 运输 + 重建）
- 资金成本（搬运费用）
- 机会成本（迁移期间停止生产）
- 风险成本（迁移途中易受攻击）

**搭建限制**：
- 矿场 → 必须在矿脉上
- 农场 → 必须在平原/水源附近
- 港口 → 必须在海岸/河流
- 机场 → 必须平坦地形

### 3️⃣ 时代范围

**由地图/战役编辑器决定**：
- 游戏本身是"战争模拟平台"
- 时代不是限制，是编辑器参数
- 期待玩家复现历史上所有战争

**官方 MOD 计划**：
- 第一步：红军战史（近代）
- 第二步：三国战史（古代）

## 🧠 个体行为三原则

### 第一原则：确保生存

**恐慌机制**：
```
恐慌度 = f(饮食剩余，距离补给点，敌方数量，己方数量，士气)

阈值：
- 30%: 焦虑，效率下降
- 60%: 士气快速下降
- 80%: 考虑逃跑/投降
- 100%: 强制溃散
```

### 第二原则：依附编制

**编制的作用**：
1. 补给来源
2. 集群威慑
3. 命令传达
4. 归属感

**群体崩溃机制**：
```
个体恐慌 → 群体恐慌 → 凝聚力下降 → 群体崩溃 → 传染扩散
```

### 第三原则：渴望晋升

**晋升机制**：
- 系统默认 100% 自动晋升（按军功/能力/资历）
- 玩家可干预（满足"当长官"趣味），但有士气代价

## 🏗️ 命令系统

### 核心原则

**玩家不能直接指挥个体**：
- ✅ 玩家只能对编制下达命令
- ✅ 命令需要传令兵送达才生效
- ✅ 传令延迟取决于距离和通讯手段

### 传令兵机制

**自动多路径传令**：
- 玩家可调整命令紧急度
- 紧急度高 → 自动多派传令兵、分路传送
- 这是战略层面考虑，**非玩家微操**

### 通讯手段演进

| 时代 | 通讯方式 | 范围 | 延迟 |
|------|---------|------|------|
| **古代** | 传令兵（骑马） | 无限 | 高 |
| **古代** | 旗鼓号 | 视距/声距 | 低（信息简单） |
| **近代** | 电台 | 无限 | 即时 |
| **现代** | 数据链 | 无限 | 即时 + 可视化 |

## 🏭 装备系统

### 核心原则

**玩家不需要攀科技树**：
- ❌ 玩家不研发科技
- ✅ 装备由编辑器预设
- ✅ 装备从经济建筑产出

### 装备分类

| 分类 | 说明 | 示例 |
|------|------|------|
| **武器类** | 攻击装备 | 刀/枪/炮/导弹 |
| **防具类** | 防御装备 | 盔甲/防弹衣 |
| **通讯类** | 通讯装备 | 旗鼓号/电台/数据链 |
| **载具类** | 移动工具 | 马匹/车辆/直升机 |
| **补给类** | 消耗品 | 粮草/燃油/弹药 |

---

## 🎓 训练系统

### 核心原则

**士兵需要训练时间 + 玩家付出资金**：
- 新兵不能直接使用复杂装备
- 训练需要时间（4-8 周基础训练）
- 训练需要资金（教官/场地/耗材）
- 熟练度可提升（Lv1 新兵 → Lv5 精英）

### 熟练度等级

| 等级 | 名称 | 效果 | 训练时长 |
|------|------|------|---------|
| Lv1 | 新兵 | 效率 50% | 0-4 周 |
| Lv2 | 合格 | 效率 75% | 1-3 月 |
| Lv3 | 熟练 | 效率 100% | 3-6 月 |
| Lv4 | 老兵 | 效率 125% | 6-12 月 |
| Lv5 | 精英 | 效率 150% | 1 年 + |

---

## 🏛️ 经济建筑系统

### 核心原则

**经济建筑 = 不会移动的军事编制**（但可迁移）

### 建筑分类

| 建筑类型 | 产出 | 时代 | 移动限制 |
|---------|------|------|---------|
| **农场** | 粮食 | 所有时代 | 平原/水源 |
| **兵营** | 新兵 | 所有时代 | 无 |
| **马厩** | 马匹 | 古代/近代 | 无 |
| **工厂** | 武器/车辆 | 近代/现代 | 无 |
| **矿场** | 矿石 | 所有时代 | 矿脉上 |
| **炼油厂** | 燃油 | 近代/现代 | 油田附近 |
| **军校** | 训练加速 | 所有时代 | 无 |
| **医院** | 药品/治疗 | 所有时代 | 无 |
| **港口** | 海运 | 所有时代 | 海岸/河流 |
| **机场** | 空军 | 现代 | 平坦地形 |

### 迁移机制

```
玩家决策 → 拆卸（时间 + 资金）→ 运输（时间 + 风险）
   → 到达 → 检查条件 → 重建（时间 + 资金）→ 恢复生产
```
