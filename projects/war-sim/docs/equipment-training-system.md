## 🎯 核心设计理念

### 玩家定位
- ✅ 玩家是**战略战争大师**，不是科技专家
- ❌ 玩家**不需要**攀科技树
- ✅ 装备/科技由**地图编辑器**或**战役编辑器**预设

### 时代差异实现

**通过装备限制实现时代差异**：
```
古代战场：
  经济建筑产出：马匹、粮草、简易武器
  通讯装备：无（只能靠传令兵/旗鼓号）
  
近代战场：
  经济建筑产出：车辆、燃油、电台、步枪
  通讯装备：电台（即时通讯）
  
现代战场：
  经济建筑产出：直升机、卫星数据链、精确武器
  通讯装备：数据链（即时 + 可视化）
```

## 🏭 装备系统

### 装备分类

| 分类 | 说明 | 示例 |
|------|------|------|
| **武器类** | 攻击装备 | 刀/枪/炮/导弹 |
| **防具类** | 防御装备 | 盔甲/防弹衣/头盔 |
| **通讯类** | 通讯装备 | 旗鼓号/电台/数据链 |
| **载具类** | 移动工具 | 马匹/车辆/直升机 |
| **补给类** | 消耗品 | 粮草/燃油/弹药/药品 |

### 装备获取

**不由玩家研发，由编辑器预设**：
```
地图编辑器设置:
  - 时代背景：古代/近代/现代
  - 可用装备列表：[刀，弓，马匹，旗鼓号]
  - 经济建筑产出：[粮草，马匹，简易武器]
  
战役编辑器设置:
  - 战役背景：淞沪会战
  - 中方装备：[步枪，手榴弹，电台（少量）]
  - 日方装备：[步枪，火炮，坦克，电台（充足）]
```

### 装备属性

| 属性 | 说明 | 示例 |
|------|------|------|
| **重量** | 影响负重力和移动力 | 重炮 vs 步枪 |
| **体积** | 影响携带数量 | 压缩饼干 vs 大米 |
| **射程** | 攻击距离 | 弓箭 100m vs 步枪 400m |
| **杀伤力** | 基础伤害 | 刀 10 vs 火炮 100 |
| **精度** | 命中率 | 狙击枪 90% vs 弓箭 60% |
| **射速** | 攻击频率 | 机枪 600 发/分 vs 步枪 10 发/分 |
| **通讯范围** | 通讯距离 | 电台 10km vs 旗鼓号 1km |
| **通讯延迟** | 信息传递速度 | 数据链 0s vs 传令兵 60s+ |

## 🎓 训练系统

### 核心原则

**士兵需要训练时间 + 玩家付出资金**：
- ✅ 新兵不能直接使用复杂装备
- ✅ 训练需要时间（不能立即形成战斗力）
- ✅ 训练需要资金（教官/场地/耗材）
- ✅ 熟练度可提升（越练越精）

### 技能分类

| 技能类型 | 说明 | 训练时间 | 训练成本 |
|---------|------|---------|---------|
| **基础战斗** | 使用基础武器 | 短 | 低 |
| **通讯操作** | 使用电台/数据链 | 中 | 中 |
| **载具驾驶** | 驾驶马匹/车辆 | 长 | 高 |
| **专业兵种** | 炮兵/医护/工兵 | 很长 | 很高 |
| **特种技能** | 侦察/狙击/爆破 | 极长 | 极高 |

### 训练流程

```
新兵入伍
   ↓
基础训练（4-8 周）
   ├── 体能训练
   ├── 武器使用
   └── 纪律培养
   ↓
分配到编制
   ↓
岗位训练（2-4 周）
   ├── 专业技能
   └── 协同作战
   ↓
形成战斗力
   ↓
持续训练（提升熟练度）
```

### 熟练度系统

**熟练度等级**：
| 等级 | 名称 | 效果 | 训练时长 |
|------|------|------|---------|
| Lv1 | 新兵 | 效率 50%，恐慌阈值低 | 0-4 周 |
| Lv2 | 合格 | 效率 75%，恐慌阈值中 | 1-3 月 |
| Lv3 | 熟练 | 效率 100%，恐慌阈值高 | 3-6 月 |
| Lv4 | 老兵 | 效率 125%，恐慌阈值很高 | 6-12 月 |
| Lv5 | 精英 | 效率 150%，恐慌阈值极高 | 1 年 + |

**熟练度提升**：
- 训练：稳定提升（消耗资金/时间）
- 战斗：快速提升（但有伤亡风险）
- 长期不训练/不战斗：熟练度下降

## 🏛️ 经济建筑系统

### 核心原则

**经济建筑 = 不会移动的军事编制**

### 建筑分类

| 建筑类型 | 产出 | 时代限制 |
|---------|------|---------|
| **农场** | 粮食 | 所有时代 |
| **兵营** | 新兵 | 所有时代 |
| **马厩** | 马匹 | 古代/近代 |
| **工厂** | 武器/车辆 | 近代/现代 |
| **炼油厂** | 燃油 | 近代/现代 |
| **军校** | 训练加速 | 所有时代 |
| **医院** | 药品/治疗 | 所有时代 |
| **电台塔** | 通讯范围 | 近代/现代 |

### 建筑属性

| 属性 | 说明 |
|------|------|
| **位置** | 固定在地图上 |
| **产出类型** | 产出的资源/装备类型 |
| **产出效率** | 单位时间产出量 |
| **库存容量** | 最大存储量 |
| **防御能力** | 是否有自卫能力 |
| **建造时间** | 新建/升级所需时间 |
| **维护成本** | 维持运转所需资金 |

### 建筑运作

```
经济建筑（如农场）:
  - 人员配置：农民×10
  - 原料输入：种子、肥料
  - 产出：粮食（100 单位/天）
  - 库存：最大 10000 单位
  - 维护：资金（10/天）
  
当库存满或玩家指定:
  - 可转换为资金（卖出）
  - 可调拨给军队（补给）
```

## ⚖️ 时代平衡

### 古代战场

**特点**：
- 通讯：传令兵/旗鼓号（延迟高）
- 机动：马匹/步行（速度慢）
- 武器：刀/弓/矛（射程近）
- 后勤：粮草（易腐烂）

**优势**：
- 不依赖复杂后勤
- 地形影响大
- 人数优势明显

**劣势**：
- 指挥效率低
- 机动性差
- 杀伤力低

### 近代战场

**特点**：
- 通讯：电台（即时，但装备有限）
- 机动：车辆/火车（速度快）
- 武器：步枪/火炮（射程远）
- 后勤：燃油/弹药（需持续供应）

**优势**：
- 指挥效率高
- 机动性强
- 杀伤力大

**劣势**：
- 依赖后勤
- 装备维护复杂
- 训练成本高

### 现代战场

**特点**：
- 通讯：数据链（即时 + 可视化）
- 机动：直升机/装甲车（极快）
- 武器：导弹/精确制导（极远）
- 后勤：复杂（多种弹药/燃油/配件）

**优势**：
- 指挥效率极高
- 机动性极强
- 杀伤力极大

**劣势**：
- 极度依赖后勤
- 装备极其复杂
- 训练成本极高

## 🎮 游戏设计意义

### 玩家决策点

**不需要考虑**：
- ❌ 研发什么科技
- ❌ 解锁什么装备
- ❌ 升级什么建筑

**需要考虑**：
- ✅ 如何分配现有资源
- ✅ 如何训练现有士兵
- ✅ 如何使用现有装备
- ✅ 如何制定战略计划

### 编辑器权限

**地图编辑器**：
- 设定时代背景
- 设定可用装备列表
- 设定经济建筑产出
- 设定初始资源

**战役编辑器**：
- 设定交战双方
- 设定双方装备配置
- 设定胜利条件
- 设定特殊规则

## 📊 示例场景

### 场景 A：古代战役（赤壁之战）

**地图编辑器设置**：
```
时代：古代（三国）
可用装备：[刀，枪，弓，马匹，战船，旗鼓号]
经济建筑：[农场，兵营，马厩，造船厂]
通讯方式：传令兵（骑马），旗鼓号（视距）
```

**玩家挑战**：
- 指挥效率低（靠传令兵）
- 机动性差（马匹/步行）
- 需要预判敌人动向（无实时情报）

### 场景 B：近代战役（淞沪会战）

**战役编辑器设置**：
```
时代：近代（1937 年）
中方装备：[步枪，手榴弹，电台（少量），火炮（少量）]
日方装备：[步枪，火炮，坦克，电台（充足），飞机]
经济建筑：[农场，兵营，工厂，炼油厂，军校]
通讯方式：电台（即时，但中方有限）
```

**玩家挑战**：
- 装备劣势（中方）
- 通讯劣势（电台少）
- 需要以弱胜强

### 场景 C：现代战役（假设）

**地图编辑器设置**：
```
时代：现代（2020 年）
可用装备：[步枪，导弹，直升机，无人机，数据链]
经济建筑：[农场，兵营，工厂，炼油厂，空军基地，卫星站]
通讯方式：数据链（即时 + 可视化）
```

**玩家挑战**：
- 复杂后勤管理
- 高科技装备运用
- 信息战/电子战

## 📋 待确认问题

### 装备层面
1. **装备损耗**：是否需要维修/更换？
2. **弹药类型**：是否区分不同弹药（如穿甲弹/高爆弹）？
3. **缴获装备**：是否可以使用缴获的敌方装备？

### 训练层面
4. **训练失败**：是否可能训练失败（如士兵不适合某岗位）？
5. **跨岗位训练**：士兵是否可以学习多种技能？
6. **训练优先级**：玩家是否可以调整训练优先级？

### 经济层面
7. **建筑破坏**：敌方是否可以摧毁经济建筑？
8. **建筑重建**：被毁建筑是否可以重建？
9. **资源交易**：是否允许玩家之间交易资源？
